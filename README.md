# About

本项目是本学期(大三上学期)《算法设计与分析 》作业题解。课程作业在计蒜客平台发布。

由于该平台有查重功能，为防止代码被他人”借鉴“,受到影响,我会在每周代码提交时间截止后,进行更新。

**************

# More

因为学院老师一再推迟提交截止时间,导致我错过了收集第一周的题目,所幸第一周的题目也是十分简单。

所以我会从第二周的算法题开始记录。

如果对某题有其他或者更好的解法,欢迎提`issue`或者`pull request`
